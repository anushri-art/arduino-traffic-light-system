# Arduino-Based Traffic Light Control System

## Overview

Designed and developed an Arduino-based Traffic Light Control System using red, yellow, and green LEDs to simulate real-world traffic signal operations. The system automates signal transitions based on programmed timing intervals and demonstrates the fundamentals of embedded systems and microcontroller programming.

## Components Used

* Arduino Uno
* Red LED
* Yellow LED
* Green LED
* 220Ω Resistors
* Breadboard
* Jumper Wires
* Arduino IDE

## Features

* Automated traffic signal sequencing
* Timed transitions between Red, Yellow, and Green LEDs
* Real-world traffic signal simulation
* Simple and cost-effective embedded system project
* Demonstrates Arduino programming and hardware interfacing

## Applications

* Traffic Signal Simulation
* Embedded Systems Learning
* Arduino Training Projects

## Skills

Arduino Programming, Embedded Systems, Circuit Design, Hardware Interfacing, C/C++ Programming, Troubleshooting
